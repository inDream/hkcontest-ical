# hkcontest-ical

Convert hkcontest's events into iCal for personal calendar.
Demo: (https://hkcontest.wlc.io)

## Usage

`node index.js [noOfDays]`

### Options

#### noOfDays
Events fetch start from frist day of current month until this option. Default is 60 days.
